# SNA Forensic Marketing Auditor

A high-density, professional intelligence suite for automating marketing audits and performance visualizations.

## Components
- **Forensic Engine (`sna_engine.js`)**: A programmatic framework for interacting with the Meta Marketing API to detect tracking failures and spend leakage.
- **Interactive Dashboard (`interactive_audit.html`)**: A high-end, bento-grid visualizer designed for executive-level performance reporting.
- **Audit Logic (`analyze_raw_csvs.js`)**: Legacy data reconciliation for offline CSV processing.

## Current Project: Take Charge Roofing
- **Status**: Audit Complete / Forensic Intelligence Active.
- **Key Findings**: 54.4% Funnel Leakage, $186.32 Tracking Failure, Atlanta DMA Geographic Waste.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure `.env` with Meta API credentials.
4. Run the engine: `node sna_engine.js`.
